INTRODUCTION HTML

Qu'est-ce que le HTML ? Pensez à un document que vous créeriez dans un traitement de texte tel que Microsoft Word ou Google Docs. Ils se composent généralement de plus d'un style. Ils utilisent différentes tailles de police pour indiquer différentes sections du texte, comme les en-têtes, les paragraphes du corps, les pieds de page, la table des matières, les légendes, etc. Contrairement aux humains qui peuvent simplement regarder un document et comprendre la différence entre un titre et un paragraphe, les ordinateurs n'ont pas cette intuition. Afin de rendre correctement une page Web, il doit être explicitement indiqué quel est chaque élément de contenu. Alors, comment ordonnons-nous exactement au navigateur d'afficher le contenu Web ? C'est là que le langage de balisage hypertexte (ou HTML en abrégé) peut s'avérer utile. HTML est le langage dans lequel la plupart des sites Web sont écrits. Il est utilisé pour créer des pages Web et assurer leur fonctionnalité. L'hypertexte définit le lien entre les pages Web. Le langage de balisage est utilisé pour définir le document texte dans une balise qui détermine la structure des pages Web.

arborescence HTML Le HTML étant le squelette de notre page Web, il doit être très organisé et bien structuré pour permettre au navigateur de l'afficher facilement. Ici, HTML est représenté comme un arbre de balises. L'élément racine de cet arbre est toujours la balise.

 JavaScript contient une bibliothèque standard d'objets tels que Array, Date, et Math, ainsi qu'un ensemble d'éléments de langage tels que les opérateurs,les structures de contrôles et les instructions.
